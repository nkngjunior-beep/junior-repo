A prototype of a web-based simulation platform that demonstrates how virtual cards works. 
The system allows users to generate virtual cards, simulate online purchases, perform currency conversions, and view transaction history — all without connecting to a real bank.

Features
User Authentication – Secure login & registration
Virtual Card Generator – Generates card number, CVV, and expiry date
Mock Online Shopping Module – Simulate real purchase flow
Multi-Currency Simulation – XAF, USD, NGN with exchange rate conversion
Transaction History – Tracks all simulated transactions
Transaction Fee Simulation – Percentage + fixed-rate deduction
Admin Dashboard – Manage users and simulated exchange rates
Responsive UI – Clean, simple and mobile-friendly interface

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Security: Password hashing (bcrypt)
Hosting: Hostinger / Local XAMPP
